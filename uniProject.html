Semester 3
PKM Karsa Cipta
Artificial Intelligence
Twitter Sentiment Analyzer

My group and I made a sentiment analyzer for processing Twitter data to gain insight on people's opinion about a certain movie.
We used python as the programming language with the help of tweepy API, textblob and matplotlib library. Users can enter the keyword or title of the movies they want to look for and the program will return the sentiment and polarity of the movies based on recent tweets of the movie.

PKM Karsa Cipta
Software Engineering
SnapMe!

My group and I made a mobile application based on Android operating system which can help users to take pictures by performing screen mirroring on another device. The application is useful for users who often travels alone and likes to take pictures, it can help users when they want to have their pictures taken by another person so the result is satisfactory.
We used android studio to develop the application. Users need to be connected through Wi-Fi or in the same network in order to start the screen mirroring process.
